GOOGLE CLOUD:
  • Client ID: xxxxx.apps.googleusercontent.com
  Onde: https://console.cloud.google.com > APIs > Credentials

CLOUDFLARE:
  • API Token: (criar com permissão "Edit Cloudflare Pages")
  • Account ID: (pegar no dashboard)
  Onde: https://dash.cloudflare.com

GITHUB:
  • Personal Access Token (opcional, para CI/CD manual)
  Onde: https://github.com > Settings > Developer settings
